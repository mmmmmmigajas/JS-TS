# Taller de Lógica Computacional: Migración JS a TS

Este proyecto contiene una serie de simulaciones de lógica de programación (Cajero, Hotel, Digiturno, etc.) 
originalmente desarrolladas en JavaScript y migradas a **TypeScript** para mejorar la robustez y el tipado de los datos.

## 🚀 Tecnologías Utilizadas
* **TypeScript**: Lenguaje principal para el tipado estático.
* **Node.js**: Entorno de ejecución.
* **TS-Node**: Para ejecutar los archivos de TypeScript directamente.

## 📂 Contenido del Proyecto
1. `hotelreservas.ts`: Sistema de gestión de habitaciones y reservas.
2. `cajeroautomatico.ts`: Simulación de transacciones bancarias.
3. `carritocompra.ts`: Manejo de inventario y carrito de ventas.
4. `digiturno.ts`: Sistema de turnos con prioridades.
5. `boletaestadio.ts`: Venta de entradas por zonas y capacidad.
